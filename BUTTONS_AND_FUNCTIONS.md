## 2. 计时器页面

### 2.1 TimerControls 组件

**文件位置**: `src/renderer/components/TimerControls.tsx`

| 按钮 | 图标 | 功能 | 条件 |
|------|------|------|------|
| 停止按钮 | ⏹️ (Square) | 停止当前计时 | 仅在非 idle 阶段显示 |
| 跳过按钮 | ✅ (CheckCircle) | 跳过当前阶段（进入下一阶段） | 仅在非 idle 阶段显示 |
| 开始/暂停按钮 | ▶️ (Play) / ⏸️ (Pause) | 开始计时 或 暂停计时 | 始终显示 |
| 完成任务按钮 | ✓ (CheckCircle2) | 标记当前任务为已完成 | 计时器 idle 且有选中任务且任务未完成 |

### 2.2 TaskSelector 组件

**文件位置**: `src/renderer/components/TaskSelector.tsx`

| 按钮 | 功能 |
|------|------|
| 任务卡片 | 点击选择该任务作为当前计时任务 |
| 取消选择按钮 | 取消当前选中的任务 |
| 显示全部 / 收起按钮 | 展开或收起任务列表 |
| 新建任务按钮 | 打开新建任务弹窗 |

## 3. 任务管理

### 3.1 任务列表 (TaskList)

**文件位置**: `src/renderer/components/TaskList.tsx`

#### 筛选按钮

| 按钮 | 功能 |
|------|------|
| 全部 | 显示所有任务 |
| 进行中 | 仅显示状态为"进行中"的任务 |
| 已完成 | 仅显示状态为"已完成"的任务 |
| 全部 (任务类型) | 显示所有任务类型 |
| AI 任务 | 仅显示 AI 任务类型 |
| 普通任务 | 仅显示普通任务类型 |

#### 任务项操作

| 按钮 | 图标 | 功能 | 位置 |
|------|------|------|------|
| 完成任务 | ⭕ (圆形选择框) | 标记任务为已完成/进行中 | 任务左侧 |
| 任务标题 | 文字区域 | 点击选中任务（打开操作菜单时） | 任务中间 |
| 开始计时 | ▶️ (Play) | 选择该任务并开始计时 | 更多菜单 |
| 编辑任务 | ✏️ (Edit2) | 打开编辑任务弹窗 | 更多菜单 |
| 删除任务 | 🗑️ (Trash2) | 删除任务（会二次确认） | 更多菜单 |

### 3.2 任务表单 (TaskForm)

**文件位置**: `src/renderer/components/TaskForm.tsx`

| 按钮 | 功能 |
|------|------|
| 关闭按钮 (X) | 关闭弹窗，取消创建/编辑 |
| 预估番茄数按钮 (1,2,3,4,5,8) | 设置任务预估番茄数 |
| + 按钮 | 番茄数 +1 |
| AI 任务按钮 | 将任务类型设为 AI |
| 普通任务按钮 | 将任务类型设为普通 |
| 重要且紧急按钮 | 设置为第一象限优先级 |
| 重要不紧急按钮 | 设置为第二象限优先级 |
| 不重要但紧急按钮 | 设置为第三象限优先级 |
| 不重要不紧急按钮 | 设置为第四象限优先级 |
| 添加日期按钮 | 添加任务的计划执行日期 |
| 删除日期按钮 (🗑️) | 移除已添加的日期 |
| 取消按钮 | 取消创建/编辑，关闭弹窗 |
| 保存/创建按钮 | 提交表单，创建或更新任务 |

## 7. 设置页面

**文件位置**: `src/renderer/components/SettingsForm.tsx`

### 7.1 计时器设置

| 输入框 | 功能 |
|--------|------|
| 番茄时长 (分钟) | 设置单个番茄钟时长 (1-60) |
| 短休息时长 (分钟) | 设置短休息时长 (1-30) |
| 长休息时长 (分钟) | 设置长休息时长 (1-60) |
| 长休息间隔 (次) | 设置长休息间隔 (1-10) |

### 7.2 行为设置

| 复选框 | 功能 |
|--------|------|
| 自动开始下一个番茄钟 | 番茄钟结束后自动开始下一个 |
| 开机自动启动 | 系统启动时自动运行应用 |
| 空闲检测 | 检测用户空闲状态 |
| 启用声音 | 休息结束时播放提示音 |
| 启用通知 | 发送系统通知提醒 |

### 7.3 语言设置

| 按钮 | 功能 |
|------|------|
| 中文 | 切换界面语言为中文 |
| English | 切换界面语言为英文 |

### 7.4 主题设置

| 按钮 | 功能 |
|------|------|
| ☀️ 浅色 | 切换为浅色主题 |
| 🌙 深色 | 切换为深色主题 |
| 💻 跟随系统 | 跟随系统主题设置 |

### 7.5 其他操作

| 按钮 | 功能 |
|------|------|
| 🔄 重置默认设置 | 重置所有设置为默认值 |
| 💾 保存设置 | 保存当前设置更改 |

## 8. 休息窗口 (BreakPage)

**文件位置**: `src/renderer/pages/BreakPage.tsx`

### 8.1 BreakOverlay 组件

**文件位置**: `src/renderer/components/BreakOverlay.tsx`

| 按钮 | 功能 | 条件 |
|------|------|------|
| 推迟休息按钮 | 推迟休息（有时间限制） | 非严格模式且还有推迟次数 |
| 跳过/结束休息按钮 | 跳过当前休息，进入下一阶段 | 非严格模式；严格模式下禁用 |

### 8.2 AI 任务弹窗

| 按钮 | 功能 |
|------|------|
| AI 任务选择框 | 勾选/取消勾选 AI 任务 |
| 确认并开始专注按钮 | 确认 AI 任务状态并开始下一个番茄钟 |

---

## 9. 全局快捷键

**在应用未获得焦点时也可使用**

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Shift + P` | 开始/暂停番茄钟 |
| `Ctrl/Cmd + Shift + S` | 跳过当前阶段 |
| `Ctrl/Cmd + Shift + O` | 显示/隐藏主窗口 |
| `Ctrl/Cmd + X` | 结束休息 |
| `Ctrl/Cmd + P` | 推迟休息 |
