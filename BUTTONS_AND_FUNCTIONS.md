## 2. 计时器页面

### 2.1 TimerControls 组件

**文件位置**: `src/renderer/components/TimerControls.tsx`

| 按钮 | 图标 | 功能 | 条件 |
|------|------|------|------|
| 停止按钮 | ⏹️ (Square) | 停止当前计时 | 仅在非 idle 阶段显示 |
| 跳过按钮 | ✅ (CheckCircle) | 跳过当前阶段（进入下一阶段） | 仅在非 idle 阶段显示 |
| 开始/暂停按钮 | ▶️ (Play) / ⏸️ (Pause) | 开始计时 或 暂停计时 | 始终显示 |
| 完成任务按钮 | ✓ (CheckCircle2) | 标记当前任务为已完成 | 计时器 idle 且有选中任务且任务未完成 |

### 2.2 TaskSelector 组件

**文件位置**: `src/renderer/components/TaskSelector.tsx`

| 按钮 | 功能 |
|------|------|
| 任务卡片 | 点击选择该任务作为当前计时任务 |
| 取消选择按钮 | 取消当前选中的任务 |
| 显示全部 / 收起按钮 | 展开或收起任务列表 |
| 新建任务按钮 | 打开新建任务弹窗 |

#### 任务优先级显示

TaskSelector 组件按优先级排序显示任务：

| 优先级 | 标签 | 颜色 | 图标 |
|--------|------|------|------|
| 重要且紧急 | 红色标签 | 红色边框 | AlertCircle |
| 重要不紧急 | 蓝色标签 | 蓝色边框 | Clock9 |
| 紧急不重要 | 黄色标签 | 黄色边框 | Clock |
| 不重要不紧急 | 灰色标签 | 灰色边框 | Square |

#### 今日计划任务

- 显示今日计划的任务（优先）
- 显示任务番茄进度条
- 显示任务所属项目颜色

### 2.3 TimerDisplay 组件

**文件位置**: `src/renderer/components/TimerDisplay.tsx`

| 元素 | 功能 |
|------|------|
| 倒计时数字 | 显示剩余时间（分:秒） |
| 进度环 | 可视化显示进度 |
| 当前阶段标签 | 显示当前是工作/短休息/长休息 |

## 4. 任务管理

### 4.1 任务页面顶部操作

**文件位置**: `src/renderer/pages/TasksPage.tsx`

#### 筛选按钮

| 按钮 | 功能 |
|------|------|
| 全部 | 显示所有任务 |
| 进行中 | 仅显示状态为"进行中"的任务 |
| 已完成 | 仅显示状态为"已完成"的任务 |
| 全部 (任务类型) | 显示所有任务类型 |
| AI 任务 | 仅显示 AI 任务类型 |
| 普通任务 | 仅显示普通任务类型 |

#### 页面操作按钮

| 按钮 | 图标 | 功能 | 条件 |
|------|------|------|------|
| 完成项目 | ✓ (Check) | 将选中的项目标记为已完成 | 选中某个进行中的项目时显示 |
| 空闲记录 | ⏱️ (Clock) | 查看空闲记录历史 | 始终显示 |
| 新建项目 | 📁 (FolderPlus) | 打开新建项目弹窗 | 始终显示 |
| 新建任务 | ➕ (Plus) | 打开新建任务弹窗 | 始终显示 |

### 4.2 任务列表 (TaskList)

**文件位置**: `src/renderer/components/TaskList.tsx`

#### 筛选按钮

| 按钮 | 功能 |
|------|------|
| 全部 | 显示所有任务 |
| 进行中 | 仅显示状态为"进行中"的任务 |
| 已完成 | 仅显示状态为"已完成"的任务 |
| 全部 (任务类型) | 显示所有任务类型 |
| AI 任务 | 仅显示 AI 任务类型 |
| 普通任务 | 仅显示普通任务类型 |

#### 任务项操作

| 按钮 | 图标 | 功能 | 位置 |
|------|------|------|------|
| 完成任务 | ⭕ (圆形选择框) | 标记任务为已完成/进行中 | 任务左侧 |
| 任务标题 | 文字区域 | 点击选中任务（打开操作菜单时） | 任务中间 |
| 开始计时 | ▶️ (Play) | 选择该任务并开始计时 | 更多菜单 |
| 编辑任务 | ✏️ (Edit2) | 打开编辑任务弹窗 | 更多菜单 |
| 删除任务 | 🗑️ (Trash2) | 删除任务（会二次确认） | 更多菜单 |

### 4.3 任务表单 (TaskForm)

**文件位置**: `src/renderer/components/TaskForm.tsx`

| 按钮 | 功能 |
|------|------|
| 关闭按钮 (X) | 关闭弹窗，取消创建/编辑 |
| 预估番茄数按钮 (1,2,3,4,5,8) | 设置任务预估番茄数 |
| + 按钮 | 番茄数 +1 |
| AI 任务按钮 | 将任务类型设为 AI |
| 普通任务按钮 | 将任务类型设为普通 |
| 重要且紧急按钮 | 设置为第一象限优先级 |
| 重要不紧急按钮 | 设置为第二象限优先级 |
| 不重要但紧急按钮 | 设置为第三象限优先级 |
| 不重要不紧急按钮 | 设置为第四象限优先级 |
| 添加日期按钮 | 添加任务的计划执行日期 |
| 删除日期按钮 (🗑️) | 移除已添加的日期 |
| 取消按钮 | 取消创建/编辑，关闭弹窗 |
| 保存/创建按钮 | 提交表单，创建或更新任务 |

### 4.4 任务优先级（四象限）

**Eisenhower 矩阵**

| 象限 | 条件 | 颜色 | 标签 |
|------|------|------|------|
| 重要且紧急 | isImportant=true, isUrgent=true | 红色 | 重要且紧急 |
| 重要不紧急 | isImportant=true, isUrgent=false | 蓝色 | 重要不紧急 |
| 紧急不重要 | isImportant=false, isUrgent=true | 黄色 | 紧急不重要 |
| 不重要不紧急 | isImportant=false, isUrgent=false | 灰色 | 普通 |

### 4.5 AI 任务

**任务类型**: `taskType: 'ai' | 'normal'`

| 类型 | 描述 | 特点 |
|------|------|------|
| AI 任务 | 人工智能相关任务 | 紫色标签显示 |
| 普通任务 | 一般任务 | 无特殊标签 |

AI 任务不参与番茄钟计时，但在休息时会弹出确认界面。

## 9. 休息窗口 (BreakPage)

## 11. 全局快捷键

**在应用未获得焦点时也可使用**

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Shift + P` | 开始/暂停番茄钟 |
| `Ctrl/Cmd + Shift + S` | 跳过当前阶段 |
| `Ctrl/Cmd + Shift + O` | 显示/隐藏主窗口 |
| `Ctrl/Cmd + X` | 结束休息 |
